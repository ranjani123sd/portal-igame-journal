# portal-igame-journal

## 项目简介

portal-igame-journal 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅用于集中存放与个人游戏相关记录、日志或工具页面，便于统一管理与访问。

## 目录说明

```
portal-igame-journal/
├── index.html          # 入口页面，提供导航或概述
├── pages/              # 存放各独立 HTML 页面
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
├── assets/             # 资源文件（样式、脚本、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

各目录用途如下：

- `pages/`：所有独立 HTML 页面均放置于此，按需命名。
- `assets/`：存放页面所需的样式表、脚本文件及图像资源。
- `index.html`：可选入口页面，用于展示页面列表或提供跳转链接。

## 页面归档说明

本仓库中的 HTML 页面均为独立、自包含的静态文件。每个页面通常用于记录特定主题或功能，例如个人游戏日志、工具界面或信息展示。

- 页面之间不存在依赖关系，可单独打开使用。
- 所有资源文件均存放于 `assets/` 目录下，便于统一管理。
- 如需添加新页面，请创建新的 HTML 文件并放置于 `pages/` 目录，同时更新 `index.html` 中的链接（如有）。
